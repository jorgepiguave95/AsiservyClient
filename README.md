# CoacUtilities-Cliente

**Frontend del sistema CoacUtilities**  
Proyecto desarrollado en React + TypeScript con Vite, Tailwind CSS v4, ShadCN UI y TanStack (Query, Router, Store).

---

# 🚀 Tecnologías utilizadas

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [ShadCN UI](https://ui.shadcn.com/)
- [TanStack Query](https://tanstack.com/query/latest)
- [TanStack Router](https://tanstack.com/router/latest)
- [TanStack Store](https://tanstack.com/store/latest)

# 📦 Instalación

```bash
# Instala Yarn como manejador de paquetes (opcional)
npm install --global yarn

# Clona el repositorio
git clone https://github.com/TiCoacmes/CoacUtilities-Cliente.git

# Entra al proyecto
cd CoacUtilities-Cliente

# Instala las dependencias
yarn install

# Entrar a VS Code desde cmd
code .

# Crea una nueva rama para trabajar en tu feature o corrección
git checkout -b nombre-de-tu-rama

# ------------- PARA SUBIR EL RAMAL CREADO --------------------------------

# Verifica todos los errores de linter que hay y los intenta solucionar
yarn lint

# Formatea los archivos del proyewcto
yarn format

# Realiza tus cambios y añade los archivos modificados
git add .

# Confirma los cambios
git commit -m "Descripción clara de los cambios"

# Sube tu rama al repositorio remoto
git push origin nombre-de-tu-rama

```
